Getting started with the Analog Discovery Studio

1. Hardware

To set up the Analog Discovery Studio:

  1. First, install the Digilent WaveForms software on your PC. See Installer Details for more information.
  2. Connect the Analog Discovery Studio to your PC using a USB cable and 15 VDC 2.4 A power supply.
